# Moss-plant-comparisons
Repository that contains all the files to reproduce the analyses in which we compare the temporal trend of the community temperature index of mosses and plants along the elevational gradient in Switzerland. 

## Content

1. **Data-raw:** This folder contains the data files used in the presented study. These files were exported from the BDM data-base as described in `R/Data-preparation.R`.
2. **DB:** This folder contains the BDM dat-base. Note, however, that this folder is not available from GitHub. 
3. **div:** Contains various files such as the file with the references (`bibliography.bib`).
4. **Manuscript:** This folder contains the Rmarkdown-File to produce the method and results sections of the manuscript (`MS.Rmd`) and the files that are produced when running this markdown file.
5. **R**: This folder contains additional R-Skripts. In [Data-preparation.R](R/Data-preparation.R) the data needed for the analyses are exported from the BDM data-base, appropriately structured and saved to `Data-raw`.
